# Vue.extend

使用基础 Vue 构造器，创建一个“子类”。

data 选项是特例，需要注意 - 在 Vue.extend() 中它必须是函数

<div id="app"></div>
1
const Name = Vue.extend({
  template: "<div>{{name}}</div>",
  data: function() {
    return {
      name: "jack"
    };
  }
});
new Name().$mount("#app");
